Morse code decoder

Take a look at the legend page to get a quick overview of the different symbols. To morse just click and release the button.
To get used to the morser keep following in mind:
- We can produce long and short signals and long and short breaks.
- If you stop morsing for more then 900ms a long break will be automatically emitted to start the translation.
